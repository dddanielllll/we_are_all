This project is a minimal web game designed to run on Vercel.

Files included:
- index.html
- styles.css
- game.js
- api/summary.js
- vercel.json
- package.json

Deployment:
- Push this repo to GitHub and import to Vercel, or use `vercel` CLI from the project folder.
- The site serves a static HTML front-end and a small serverless JSON endpoint at `/api/summary`.

Gameplay:
- Mix of narrative scenes and two short mini-games (matching + ordering).
- Suitable for ages 8+, no violence, educational focus on empathy and anti-discrimination.
